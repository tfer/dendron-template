# Dendron Next Server

## Quickstart
1. follow instructions [[here|dendron.dev.setup]] to clone the repository and install dependencies
1. navigate to server package
    ```sh
    cd {repo-root}/packages/dendron-next-server
    ```
1. start server
    ```sh
    yarn dev
    ```
